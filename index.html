Command line weather

cloudburst is the result of some slacking off and learning new things along the way.

What isn't uploaded to github (Yet) are a TON of Markdown notes stored in a VimWiki on how to do alot of things. These notes are what helped me try to integrate what I've learned over the past few months into a practical mnemonic resource of vital information.

Over time, I found myself using Vim more than VSCode and IntelliJ. The nuances of using Regular Expression and trying to transform information quickly and with minimal effort became more of an interest than using some clunky graphic user interface.

In the text-base environment I felt more at home. Like it was learning how to play the piano as opposed to using a drum machine. Sure, it's cool, but it's no fun if there is no creativity and imagination at work.

Command line programs will always have a place in the heart of any computer programmer devoted to their work. It's the difference between writing a book and directing a movie based upon a book where a lot of stuff get's cut out.

In that same spirit, made this program to include many features. Mostly stupid tricks but the important ones do a ton of cool stuff.

A better documentation of cloudburst can be found on its Github page. Some of that content will eventually make its way to this page.

Cloudburst provides the data that Blitz needs to function. Think of Blitz as the Christian de Neuvillette to Cloudburt's Cyrano de Bergerac. One project is the pretty face but the brains and the heart of this project is this side of the project.
